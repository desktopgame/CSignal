コルーチンの実装に関するメモ

# コルーチン
コルーチンとは、メソッドを途中で中断して  
次に呼び出された時に最後に中断された箇所から再開する機能。  
beaconではC#の仕様を参考にしている。

# 実装
````
*--------*
| iterae |
*--------*
     |
*--------------* [n]  *--------------------------------------*       *-----------------------------------*
| if call first| ---- | set a PC on last interrupted position| ----- | copy `ref_stack` and `value_stack`|
*--------------*      *--------------------------------------*       *-----------------------------------*
     | [y]                                                                      |
*--------------------------------*                                              |
| create and set a `yield_context|                                              |
*--------------------------------*                                              |
     |                                                                          |
     |--------------------------------------------------------------------------*
     |
     |
*-----------------*
| run statement...|
*-----------------*
     |
     |     *-------------*     *-----------------------------------------*
     *-----| yield return| --- | save `PC`, `ref_stack` and `value_stack`|
     |     *-------------*     *-----------------------------------------*
     |                                                            |
     |                                                            |
     |     *------------*                *----------*             |
     *-----| yield break|   -------->    | terminate| <-----------*
     |     *------------*                *----------*
     |                                        ^
     |                                        |
     |     *--------------------*             |
     *-----| finished statement | ------------*
           *--------------------*
````
