# beacon
beacon(ビーコン)は静的型付けのプログラミング言語です

# ファイル
* bin --成果物
* obj --中間生成されるオブジェクトファイル
* src --ソースコード

# ブランチ
* **develop** が 開発中のブランチ
* **windows** が windows用ブランチ
* **master** が 最後のリリース
